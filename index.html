Handshaking: Σ deg(v) = 2m
Simple graphNo loops, ≤1 edge/pair

deg⁺(v)Out-degree (edges leaving)
deg⁻(v)In-degree (edges entering)

BipartiteEdges only cross partitions
No odd cycles⟺ bipartite
Complete matchingEvery V₁ node matched
K_{m,n} edgesm × n

🌉 Euler Paths

Euler CYCLE ↔ all even degrees
Euler PATH ↔ exactly 2 odd degrees
Euler cycleEvery edge once, return to start
Directed Euler cycledeg⁺(v) = deg⁻(v) for all v
Königsberg4 odd vertices → no solution

🏇 Hamiltonian

Hamiltonian cycleEvery VERTEX exactly once
Euler vs HamiltonEdges vs Vertices
Dirac: n≥3, all deg ≥ n/2 → Ham. cycle
Ore: deg(u)+deg(v) ≥ n → Ham. cycle
NoteNo simple iff condition (NP-hard)

🎲 Probability

P(A∪B) = P(A)+P(B)−P(A∩B)
P(A|B) = P(A∩B) / P(B)
Independent: P(A∩B) = P(A)·P(B)
ComplementP(Ā) = 1 − P(A)
Disjoint ≠ IndependentDon't confuse these!
Bonferroni: P(A∩B) ≥ P(A)+P(B)−1

🔮 Bayes' Theorem

P(A|B) = P(A)·P(B|A) / P(B)
P(B) = P(A)·P(B|A) + P(Ā)·P(B|Ā)
P(A) = PriorBefore seeing evidence
P(A|B) = PosteriorAfter seeing evidence
P(B|A) = LikelihoodProbability of evidence given A

📊 Binomial Distribution

P(X=k) = C(n,k)·pᵏ·(1−p)ⁿ⁻ᵏ
E[X] = np
C(n,k) = n! / (k!(n−k)!)
C(6,3)20
C(6,4)15
C(6,5)6
