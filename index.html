📋 Quick Cheatsheet

🕸️ Graph Basics

Simple graphNo loops, ≤1 edge per pair
MultigraphMultiple edges allowed
deg(v)Number of edges at v
Pendant nodedeg = 1
Isolated nodedeg = 0
Neighbourhood N(v)All adjacent nodes
Handshaking: Σ deg(v) = 2m

📐 Directed Graphs

deg⁺(v) out-degreeEdges leaving v
deg⁻(v) in-degreeEdges entering v
LoopCounts +1 to both deg⁺ and deg⁻
Σ deg⁺(v) = Σ deg⁻(v) = m

🔗 Bipartite Graphs

Bipartite G=(V₁∪V₂,E)Edges only cross partitions
Matching MNo two edges share a node
Complete matchingEvery node in V₁ matched
Maximum matchingLargest possible matching
Hall's Theorem: ∀A⊆V₁, |N(A)| ≥ |A|
Hall's conditionComplete matching exists iff condition holds

🔀 Isomorphism

IsomorphismBijection f: V₁→V₂ preserving edges
Quick disproofsDifferent # of nodes/edges
Different degree sequences
Induced subgraphKeep selected nodes + all edges between them

🔌 Connectivity

Path (length n)Sequence of n edges
Simple pathNo edge repeated
CyclePath starting and ending at same node
Connected (undirected)Path between every pair
Strongly connectedDirected path a→b AND b→a for all pairs
Weakly connectedUnderlying undirected graph is connected
Connected componentMaximal connected subgraph

🌉 Euler Paths

Euler cycleSimple cycle using EVERY edge
Euler pathSimple path using EVERY edge
Euler cycle exists ↔ all vertices even degree
Euler path exists ↔ exactly 2 odd-degree vertices
Directed Euler cycledeg⁺(v) = deg⁻(v) for ALL v

🏇 Hamiltonian Paths

Hamiltonian cycleVisits every VERTEX exactly once
Euler vs HamiltonEuler = edges, Hamilton = vertices
Dirac: n≥3, all deg≥n/2 → Ham. cycle
Ore: n≥3, deg(u)+deg(v)≥n for non-adjacent u,v → Ham. cycle
NoteNo simple condition like Euler

🎲 Probability

P(A) = |A| / |S|
P(Ā) = 1 - P(A)
P(A∪B) = P(A)+P(B)-P(A∩B)
P(A|B) = P(A∩B) / P(B)
Independent eventsP(A∩B) = P(A)·P(B)
Bonferroni: P(E∩F) ≥ P(E)+P(F)-1

🔮 Bayes' Theorem

P(A|B) = P(A)·P(B|A) / P(B)
Full: P(A|B) = P(A)·P(B|A) / [P(A)·P(B|A) + P(Ā)·P(B|Ā)]
P(A)Prior probability
P(A|B)Posterior probability
P(B)Marginal probability
P(B|A)Likelihood

📊 Distributions

Bernoulli trialSingle experiment, p=success
Binomial B(n,p)n independent Bernoulli trials
P(X=k) = C(n,k)·pᵏ·(1-p)ⁿ⁻ᵏ
Expected value E[X]np
C(n,k)n! / (k!(n-k)!)
